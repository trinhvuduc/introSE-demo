## Đầu tiên, cài đặt Nodejs

https://nodejs.org/en/

## Sau đó, cài đặt vài thư viện cần thiết

```
cd client
npm install
cd ..
cd server
npm install
```

## Tiếp theo, mở terminal trỏ về đúng folder

## Run client

```
cd client
npm start
```

## Run server

Mở một terminal khác

```
cd server
npm start
```

Enjoy ^^
